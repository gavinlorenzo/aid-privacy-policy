AID Privacy Policy

Effective date: March 5, 2026

AID ("we", "our", "the extension") is a browser extension that builds a behavioral personality profile from your interactions with AI chat platforms (ChatGPT and Claude). This profile is used to personalize how those AI assistants respond to you.

This policy explains what data we collect, how we use it, who we share it with, and how you can control or delete it.

1. What Data We Collect

Data Purpose Stored where
Email address and password Account authentication Supabase Auth (server-side)
Chat message text (up to 1,200 characters per message) Sent to an AI model for behavioral pattern extraction. Not stored in our database. Transmitted to LLM, not persisted
Behavioral observations (e.g., "Prefers numbered steps") Core profile data — describes how you think and communicate Supabase database (server-side)
Short evidence quotes (brief excerpts from your messages) Support and validate each behavioral observation Supabase database (server-side)
Conversation context snippets (up to 1,200 characters) Provide context around each observation at creation time Supabase database (server-side)
Conversation IDs (platform-assigned identifiers) Track which conversations observations came from Supabase database (server-side)
Auth session tokens Keep you logged in Browser local storage (your device)
Conversation injection state (metadata only — no message text) Track whether your profile was already injected into a conversation Browser local storage (your device)
What we do NOT collect: We do not store your full chat messages or conversation transcripts. We do not collect browsing history, cookies, personal files, or any data from sites other than ChatGPT and Claude. We do not use your data for advertising.

2. How We Use Your Data

  • Profile building: Your chat messages are analyzed in real time by an AI model to extract behavioral patterns — how you communicate, learn, and solve problems. Only the resulting observations are stored, not the messages themselves.
  • Profile injection: Your behavioral profile is prepended to your messages on supported platforms so the AI assistant can adapt its responses to your style.
  • Profile consolidation: Periodically, raw observations are synthesized into higher-confidence patterns by an AI model. This is capped at 10 runs per user per 30-day period.

3. Third-Party Services

We use the following third-party services to operate AID:

Service Purpose Data shared
Supabase Authentication, database, server functions Email, password (auth only), behavioral observations, evidence quotes, context snippets
OpenRouter AI model API gateway Message text (for analysis), behavioral observations
DeepSeek (via OpenRouter) AI model for behavioral extraction and consolidation Message text (for analysis), behavioral observations
Google Fonts UI typography Your IP address (standard web font loading)

We do not sell, rent, or trade your personal data to any third party.

4. Data Security

  • All data in transit is encrypted via HTTPS/TLS.
  • Database access is enforced by Row Level Security — each user can only access their own data.
  • Authentication tokens are stored in your browser's local storage and are never shared with third parties.
  • Server-side functions authenticate every request with a valid user token before processing.

5. Data Retention and Deletion

  • Your behavioral profile data is retained as long as your account is active.
  • You can reset your profile at any time from the AID dashboard, which deletes all observations and your profile summary.
  • You can delete individual observations from the dashboard at any time.
  • To delete your account entirely, contact us at the email below.

6. Permissions

The extension requests the following browser permissions:

  • storage: Save your login session and conversation state locally on your device.
  • activeTab: Interact with the currently active tab when it is on a supported platform.
  • Host access (chatgpt.com, chat.openai.com, claude.ai): Read and modify page content on these specific sites to intercept chat messages and inject your profile.

The extension does not request access to any other websites, your browsing history, bookmarks, downloads, or any other browser data.

7. Children's Privacy

AID is not intended for use by anyone under the age of 13. We do not knowingly collect data from children under 13.

8. Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated effective date. Continued use of the extension after changes constitutes acceptance of the updated policy.
